注册 Github 账号

如果你已经有一个Github账号可以跳过此步。

打开链接 Github 注册一个账号。用户名需要用心想一个，因为在你没有自己的独立域名的情况下，你的用户名将成为你博客地址的一部分(username.github.com）

Github 建一个新的代码仓库

点击头像，然后点New按钮进入新建代码仓库（repositories）的页面。Repository name 一栏输入仓库名称。Github用一个仓库来组织你的GitPage 资源。待会我们的网页源文件都会放到这个仓库当中。由于这个用于GitPage 的特殊性，它需要满足如下命名规则 username.github.io 其中username替换为自己的用户名。我的GitHub用户名是landuofeng，所以我的Repository name就填landuofeng.github.io

之后第一次输入提示是请求输入一个路径用于保存生成的密钥，默认是C:/users/username/.ssh，一般默认即可，所以直接按Enter建。然后是输入密码，及确定密码的提示。默认为空，如此时输入密码，则改密码将用于提交本地代码到Github时进行身份验证，默认则按Enter建。

用文本编辑器打开刚刚生成的密钥文件中的 id_rsa.pub 复制其中的全部内容。然后打开Github，进入设置。点开 SSH and GPG keys 添加新的 SSH key. 完成后回到控制台进行测试。
